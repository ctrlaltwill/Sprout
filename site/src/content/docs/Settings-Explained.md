Settings Explained is the shortcut page for picking the right LearnKit settings tab.

Use it when you know what you want to change, but not where LearnKit put that control.

## Start Here If You Want To...

- change the look of LearnKit, your greeting, your name, or Reading view styling: go to **General**
- change text to speech, default voices, or flag-based voice routing: go to **Audio**
- set up providers, models, API keys, prompts, privacy, or AI flashcard generation: go to **Companion**
- move attachment folders, store databases in the vault, create backups, or tune analytics coaching: go to **Data & Maintenance**
- change cloze behaviour, image occlusion reveal mode, multiple-choice shuffle, ordered-question shuffle, or parsing rules such as delimiters: go to **Flashcards**
- control which notes are eligible for note review or change note-review scheduling: go to **Notes**
- adjust reminders, launch nudges, Gatekeeper timing, or Gatekeeper bypass: go to **Reminders**
- change daily study limits, grading layout, sibling handling, flashcard scheduling, or FSRS optimisation: go to **Studying**
- reset settings, wipe analytics, reset scheduling, or delete all flashcards: go to **Reset**

## Quick Map By Tab

| Tab | Open it when you want to... |
|---------|-------------|
| General | Change theme, accent, animations, greeting text, name, interface language, or Reading view card style |
| Audio | Configure text to speech, autoplay, voice selection, script fallbacks, speech rate, or pitch |
| Companion | Pick an AI provider, select a model, add API keys, choose privacy limits, or tune generator output |
| Data & Maintenance | Manage attachment folders, vault database copies, backups, and analytics-coaching thresholds |
| Flashcards | Change flashcard-type behaviour and parsing rules such as code-fence handling and delimiters |
| Notes | Filter note-review scope and choose FSRS or LKRS note scheduling |
| Reminders | Control launch reminders, routine reminders, Gatekeeper popups, and bypass rules |
| Studying | Tune study-session defaults, flashcard scheduling presets, and FSRS optimisation |
| Reset | Run reset actions or destructive cleanup actions |

## Common Changes

| If you want to... | Open | Then look for... |
|---------|-------------|-------------|
| change the flashcard look in Reading view | General | Card styling, Card style, and the Reading view field toggles |
| make LearnKit speak only one subset of cards | Audio | Limit to group |
| make flags control accent or language | Audio | Use flags for language and accent |
| keep AI chats per note | Companion | Save chat history |
| send fewer linked notes or attachments to AI | Companion | Linked notes context limit, Text attachment context limit, and the Companion or Test source toggles |
| generate fewer cards or disable one generated card type | Companion | Target number of cards and Flashcard types to generate |
| move image occlusion masks or card media to another folder | Data & Maintenance | Image occlusion folder and Card attachment folder |
| keep database files inside the vault for Obsidian Sync | Data & Maintenance | Store databases in vault and Vault sync folder |
| make a restore point before a risky change | Data & Maintenance | Create manual backup |
| hide flashcards inside fenced code blocks | Flashcards | Ignore fenced code blocks |
| change the delimiter used in card markup | Flashcards | Card delimiter |
| exclude folder notes from note review | Notes | Skip folder notes |
| review only notes from certain folders, tags, or properties | Notes | Include or exclude notes |
| use LKRS instead of FSRS for note review | Notes | Scheduling algorithm |
| change daily study load | Studying | Daily new limit and Daily review limit |
| switch between two-button and four-button grading | Studying | Grading buttons |
| bury or disperse sibling flashcards | Studying | Sibling card management |
| change flashcard retention targets or learning steps | Studying | Preset, Learning steps, Relearning steps, Requested retention |
| make Gatekeeper less aggressive | Reminders | Gatekeeper interval, Questions per popup, Gatekeeper scope, and Pause gatekeeper while studying |
| clear analytics without deleting scheduling | Reset | Reset analytics |
