# Reading View Styles

LearnKit currently exposes two Reading View styles in `Settings -> General -> Reading view`.

## Available now

| Style | Layout | Default emphasis | Controls |
|------|--------|------------------|----------|
| Flashcards | Masonry | fast recall | show edit button, show audio button |
| Clean markdown | Vertical | readable note-style review | show or hide title, question, options, answer, info, groups, and labels |

## Flashcards

Flashcards style is the compact study-first option.

- Uses a flip-card presentation.
- Hides extra chrome by default.
- Keeps optional inline audio and edit actions.
- Works well when you want many cards visible in a small space.

## Clean markdown

Clean markdown is the note-first option.

- Uses a vertical layout.
- Keeps the content readable without the flip-card feel.
- Lets you decide which flashcard fields are visible.
- Is better for proofreading and content-heavy notes.

## Not currently selectable

The reading engine still contains older preset concepts such as `Classic`, `Guidebook`, and `Custom`, but the live settings UI only exposes Flashcards and Clean markdown right now.

## Choosing between them

- Choose Flashcards when recall speed matters more than full context.
- Choose Clean markdown when you want to inspect field structure or read cards in note order.

For the future-facing custom hooks, see [Custom-Reading-Styles](./Custom-Reading-Styles).
