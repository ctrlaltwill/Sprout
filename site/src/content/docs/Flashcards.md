Flashcards are the core active-recall workflow in LearnKit.

If you want the shortest path from notes to review, this is the part of the docs you will use most.

## What Flashcards Covers

Flashcards in LearnKit includes:

- creating flashcards from notes
- formatting flashcards correctly
- editing flashcards later
- reviewing flashcards in study sessions
- browsing and managing flashcards in bulk

## Typical Flashcard Workflow

1. write or refine a note
2. create flashcards from that note
3. [Sync](../Syncing) your database
4. review the flashcards in [Study Sessions](../Study-Sessions)
5. improve the source note or flashcard wording over time
