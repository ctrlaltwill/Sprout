title: "Study Widget"
---


The LearnKit widget is the quick-review sidebar for the note you are currently working in.

It is best for small, local review bursts without leaving your note context.

## Open Widget

Open it from the LearnKit sidebar icon or the command that opens the study widget.

## How Scope Works

The widget is not the same as the [Flashcard Library](../Flashcard-Library).

It usually works from the **current note**.

That means:

- on a normal note, the widget uses cards from that note
- on a folder note, it can use the folder and subfolder cards when **Treat folder notes as decks** is enabled in **Settings -> Studying**

## Summary Mode

Before you start, the widget shows a quick summary for the current scope.

That summary can include:

- new card count
- learning card count
- review card count
- due card count
- estimated study time
- **Start Studying** and **Start Practice** actions

## Session Mode

Once you start, the widget switches into a one-card-at-a-time review session.

Scheduled sessions update card scheduling normally.

Practice sessions do **not** change scheduling.

## Practice Sessions

Use **Start Practice** when you want to run through the current scope without affecting review dates.

Practice mode is useful when:

- you want a warm-up run
- you are checking cards before an exam
- you want extra exposure without changing the schedule

## Folder Notes As Decks

If your vault uses folder notes, the widget becomes more useful when **Treat folder notes as decks** is on.

In that setup, opening the folder note lets the widget treat that folder as one study scope.

## When To Use The Widget

Use the widget when you want:

- quick reviews from the note you are already in
- a sidebar study tool that stays close to your writing workflow
- a fast practice run without opening a larger study surface

Use the [Flashcard Library](../Flashcard-Library) when you need bulk management, browsing, or bigger vault-wide review control.
