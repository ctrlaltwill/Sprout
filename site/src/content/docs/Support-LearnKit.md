# Support LearnKit

LearnKit is an open-source project built to keep spaced repetition, note review, and study workflows inside your Obsidian vault.

The project is intended to stay accessible, local-first, and free to use.

## Report bugs or request features

If something is broken, unclear, or missing, GitHub issues are the best place to report it.

- [Report a bug](https://github.com/ctrlaltwill/LearnKit/issues/new?template=bug_report.yml)
- [Request a feature](https://github.com/ctrlaltwill/LearnKit/issues/new?template=feature_request.yml)
- [Browse open issues](https://github.com/ctrlaltwill/LearnKit/issues)

## Ways to support the project

If LearnKit is useful to you, the most helpful support is simple:

- star the repository
- share LearnKit with people who would use it
- report bugs clearly
- request improvements with concrete examples
- sponsor development if you want to support the work directly

## Project model

LearnKit does not use a locked subscription model for core studying.

Companion uses your own provider key instead of adding a LearnKit markup layer. That keeps provider choice in your hands and lets free-tier workflows stay possible when providers allow them.

See [Guide for Free Usage](./Guide-for-Free-Usage) and [AI Usage Policy](./AI-Usage-Policy) for more on that side of the app.
