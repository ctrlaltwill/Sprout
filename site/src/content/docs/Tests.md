title: "Tests"
---


Tests helps you generate structured question rounds from your notes and card content.

Use it when you want active recall in a more exam-like format than standard flashcard review.

## Typical Workflow

1. choose a scope such as a note, folder, or wider deck context
2. generate a test set
3. answer the questions
4. review weak areas
5. turn repeated weak spots into better notes or better cards

## When Tests Are Most Useful

Tests works well for:

- consolidation after flashcard review
- checking topic readiness before an exam
- finding weak areas that are easy to miss in normal review

## Good Practice

- keep tests topic-specific when possible
- use Coach if you want tests tied to a study plan
- pair tests with [Analytics](../Analytics) if you want to track trends over time
