# Keyboard Shortcuts

Keyboard shortcuts are the fastest way to create clozes and move through reviews.

## Cloze Creation

### In the cloze modal

| Shortcut | Action |
|---|---|
| Cmd+Shift+C (Mac) / Ctrl+Shift+C (Win/Linux) | Create a new cloze with the next number |
| Cmd+Shift+Option+C (Mac) / Ctrl+Shift+Alt+C (Win/Linux) | Create a cloze using the current number |

### In Markdown source view

| Shortcut | Action |
|---|---|
| Cmd+Shift+C (Mac) / Ctrl+Shift+C (Win/Linux) | Insert a cloze wrapper using `c1` |

## Study Sessions

| Key | Action |
|---|---|
| `Space` / `Enter` | Reveal the answer |
| `1` - `2` in two-button mode | Again / Good |
| `1` - `4` in four-button mode | Again / Hard / Good / Easy |
| `E` | Edit the current flashcard. See [Editing Flashcards](./Editing-Flashcards). |
| `O` | Open the flashcard in its source note |
| `M` | Open the More menu |
| `Q` / `Escape` | Exit the session or go back to deck selection |
| `B` | Bury the current flashcard. See [Burying Flashcards](./Burying-Flashcards). |
| `S` | Suspend the current flashcard. See [Suspending Flashcards](./Suspending-Flashcards). |
| `U` | Undo the last grade |
| `ArrowLeft` | Go to the previous flashcard or hide the answer |

## Multiple Choice Input

| Key | Action |
|---|---|
| `1` - `9` | Select an MCQ option by number |

## General

| Key | Action |
|---|---|
| Cmd/Ctrl + P | Open Obsidian's command palette |

## Remapping Shortcuts

LearnKit commands are registered through Obsidian.

Open **Settings -> Hotkeys** and search for `LearnKit` to change them.
