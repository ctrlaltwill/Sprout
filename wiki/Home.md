# Sprout

Sprout is an Obsidian plugin for writing flashcards directly inside Markdown notes and reviewing them with an FSRS-based scheduler.

## Getting started

1. [[Installation]]
2. [[Writing Cards]]
3. [[Study Sessions]]
4. [[Card Browser]]
5. [[Analytics]]
6. [[Settings]]

## Quick start

1. Write cards in any note using pipe-delimited syntax (see [[Writing Cards]])
2. Sync to create/update cards — anchors are added automatically
3. Open a study session, choose a deck or scope, and review
