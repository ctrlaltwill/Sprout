# Changelog

All notable changes to Sprout will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.3] - 2026-02-08

### Fixed
- Maintenance release - fixes and updates to Sprout settings page.

## [1.0.2] - 2026-01-XX

### Changed
- Maintenance release - code refinements prior for submission to Obsidian community plugins.

## [1.0.1] - 2025-XX-XX

### Added
- Anki import/export functionality (experimental).

### Fixed
- Stability hotfixes.

## [1.0.0] - 2025-XX-XX

### Added
- First stable release - code refactored and source shared publicly on Github.

## [0.04] - 2024-XX-XX

### Added
- Scheduling data backups.

### Changed
- Improved CSS styling (dark mode and reading-view).

## [0.03] - 2024-XX-XX

### Added
- Rebranded as Sprout.
- Image occlusion cards.
- Analytics dashboard.

## [0.02] - 2024-XX-XX

### Fixed
- Fixes focused on deck and session stability.

## [0.01] - 2024-XX-XX

### Added
- First beta release (named Boot Camp).
- Core flashcard functionality.
- FSRS-based spaced repetition.
- Basic card types (cloze, basic, multiple choice).
